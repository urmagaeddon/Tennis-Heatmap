# Simple SwingVision - Player Tracking System

A lightweight computer vision application for tracking and analyzing player movements in sports videos using YOLOv8.

## Features
- Real-time player detection and tracking
- Heatmap visualization of player movements
- Support for two-player tracking
- JSON and NumPy data export for analysis

## Requirements
- Python 3.8+
- OpenCV
- Ultralytics YOLO
- NumPy
